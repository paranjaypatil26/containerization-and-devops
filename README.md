## Overview

This repository contains practical implementations and documentation for Containerization and DevOps experiments. Each experiment focuses on applying core concepts through hands-on tasks.

The key areas covered include:

* Docker installation and setup
* Image creation and management
* Running and managing containers
* Building custom Docker images
* Comparing image sizes and layers
* Container lifecycle operations
* Port mapping and networking
* Volume mounting and data persistence
* Automation using Ansible
* CI/CD integration using Jenkins and SonarQube
